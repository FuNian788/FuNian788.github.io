STL主要内容

C++标准模板库(standard template library, STL)
C++ STL一共提供了六大组件，包括容器，算法，迭代器，仿函数，配接器和配置器。从刷题角度出发，只介绍容器和算法。

容器分为顺序容器和关联容器，顺序容器有vector，list，deque等，关联容器有set，map等。

顺序容器是按照元素在容器中的位置来保存和访问：

vector作为顺序容器，可理解为大小可变的动态数组。擅于按索引访问元素和在末尾增删元素。

deque即double-ended queue，擅于在两端增删元素。

list是一个带头结点的双向非循环链表，擅于增删元素和访问两端的元素。

关联容器是按照元素的关键字来保存和访问的：

C++中的set类似Python中的集合set，不能有重复元素，multiset则是允许有重复元素的排序好的集合。二者均不能直接修改容器中元素的值，如要更改，应先删除再插入。

C++中的map类似于Python中的字典dict，字典内无重复元素，且每个元素都由关键字和值两部分构成，multimap则是允许有重复元素的

，vector，queue， deque,，set， map

STL作用

集合set是一个无序的无重复元素序列，可用{}或者set()创建集合，如a={1,2,3}或a=set([1,2,3])或a=set((1,2,3,3))。常用操作如下：
增：a.add(x)添加单个元素；a.update(x)添加元素，参数可以为列表、元组或字典；如果元素已存在则无事发生。
删：a.remove(x)删除单个元素，如果元素不存在即报错；a.discard(x)删除单个元素，元素不存在也不报错，a.pop()删除单个随机元素。
可用len(a)计算集合内元素个数，a.clear()清空集合。
